### Usage:

```
$ page-loader -h
usage: page-loader [-h] [-o OUTPUT] url

Load Page

positional arguments:
  url

optional arguments:
  -h, --help                   show this help message and exit
  -f OUTPUT, --output OUTPUT   output directory
```
