### Usage:

```
$ page-loader -h
usage: page-loader [-h] [-f FORMAT] first_file second_file

Generate diff

positional arguments:
  first_file
  second_file

optional arguments:
  -h, --help                   show this help message and exit
  -f FORMAT, --format FORMAT   set format of output: json, plain, pretty

```
